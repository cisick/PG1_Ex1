## Merkzettel:

Für range() lautet die Formel für die Anzahl der Durchgänge: 

**(Endzahl - Anfangszzahl) : Schrittweite**    

Beispiel:  
`for j in range(0, 3, 1)`  
(3 - 0) : 1 = 3
